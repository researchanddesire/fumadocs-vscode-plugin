# Code blocks

Fenced code blocks are wrapped in the default `pre` → `CodeBlock` component with copy button and Shiki themes (`github-light` / `github-dark`).

## Basic highlighting

```tsx
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10));
```

## Line highlights and word marks

Rehype Code annotations (when supported by the compiler preset):

```ts title="config.ts"
import { createCompiler } from '@fumadocs/mdx-remote';

const compiler = createCompiler({
  rehypeCodeOptions: {
    themes: { light: 'github-light', dark: 'github-dark' },
  },
}); // [!code highlight]

export { compiler }; // [!code word:compiler]
```
