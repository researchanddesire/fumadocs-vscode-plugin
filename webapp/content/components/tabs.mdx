# Tabs

`Tabs` and `Tab` are injected globally by the preview app (imports stripped at compile time).

## Shared group (session persistence)

<Tabs groupId="package-manager" items={['npm', 'pnpm']}>
  <Tab value="npm">

```bash
npm install alpha
```

  </Tab>
  <Tab value="pnpm">

```bash
pnpm add alpha
```

  </Tab>
</Tabs>

<Tabs groupId="package-manager" items={['npm', 'pnpm']}>
  <Tab value="npm">

```bash
npm install beta
```

  </Tab>
  <Tab value="pnpm">

```bash
pnpm add beta
```

  </Tab>
</Tabs>

Selecting a tab in one block should sync the other block above (same `groupId`).
