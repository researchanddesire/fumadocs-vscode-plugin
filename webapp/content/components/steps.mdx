# Steps

Use `Steps` / `Step` for tutorials and setup guides.

## Component API

<Steps>
<Step>

### Install dependencies

```bash
pnpm install
```

Make sure Node 22+ is active (`node -v`).

</Step>

<Step>

### Configure environment

Copy `.env.example` to `.env.local` and fill in Supabase keys.

</Step>

<Step>

### Start the preview

Run **Fumadocs: Open Preview** from the command palette while editing any file in this content root.

</Step>
</Steps>
