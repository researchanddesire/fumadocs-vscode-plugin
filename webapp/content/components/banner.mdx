# Banner

`Banner` renders a dismissible announcement strip. It can adjust layout spacing when `changeLayout` is true (default).

<Banner id="docs-ohai-preview-banner">

**Preview mode** — You are viewing test content for the Fumadocs VS Code extension.

</Banner>

<Banner id="docs-ohai-normal-banner" variant="normal">

<Callout type="info">

Banners affect page layout when `changeLayout` is enabled. Scroll this page to confirm spacing above the main docs content looks correct with multiple banners present.

</Callout>
