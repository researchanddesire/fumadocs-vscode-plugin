# Accordions

Collapsible sections for FAQ-style content.

## Single open (default)

<Accordions type="single">
  <Accordion title="What is docs-ohai?">

A minimal content root used to test Fumadocs rendering in VS Code preview.

  </Accordion>
  <Accordion title="Do imports work in MDX?">

The preview strips ESM imports and injects Fumadocs components globally, so `<Callout>` and friends work without explicit imports.

  </Accordion>
  <Accordion title="How do I add a page?">

Create a new `.mdx` file under `content/` and optionally register it in a nearby `meta.json`.

  </Accordion>
</Accordions>
